Project 1: EDA Tool With Pandas Profiling and Streamlit

This program uses pandas profiling with streamlit to help speed up Exploratory Data Analysis (EDA) It allows the user to select either csv or a database as the source of data. EDA is the stage in any data Analytics activities where we examine the data in order to identify features that need further probing. It tries to capture the structer, centrality, variance etc of the data in order to advice further investigation.

it tries to do, among other things, the following: 1. maximize insight into a data set 2. uncover underlying structure 3. extract important variables 4. detect outliers and anomalies 5. test underlying assumptions
